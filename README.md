# SMART BUDGET

## Overview

SMART BUDGET is an Android mobile budgeting application developed for the OPSC6311 Portfolio of Evidence (POE).

The purpose of the application is to assist users in managing their personal finances by recording expenses, organising spending into categories, monitoring budgeting goals, visualising spending trends, and encouraging responsible financial behaviour through gamification.

The application stores all information locally using SQLite, allowing users to manage their finances without requiring an internet connection.

---

# Main Features

## User Registration and Login

SMART BUDGET allows users to create an account and securely log into the application.

Users are able to:

* Register a new account
* Log into an existing account
* Maintain their own budgeting information

---

## Expense Management

Users can record expenses by capturing:

* Expense amount
* Expense date
* Expense description
* Expense category

This allows users to keep a detailed record of their spending activities.

---

## Category Management

The application allows users to create and manage spending categories.

Examples include:

* Food
* Transport
* Entertainment
* Utilities
* Shopping

Categories help users organise expenses and improve spending analysis.

---

## Monthly Budget Goals

SMART BUDGET enables users to define:

* Minimum monthly spending goals
* Maximum monthly spending goals

These goals are used to monitor spending behaviour and assist users in staying within budget.

---

## Expense Reporting

Users can generate reports for selected periods.

Reports provide:

* Category spending totals
* Spending summaries
* Financial insights over a chosen period

This feature assists users in understanding spending patterns and making informed financial decisions.

---

## Spending Graphs

The application provides visual spending graphs.

Graphs display:

* Spending trends over time
* Budget goal comparisons
* Expense distribution information

The graph feature provides a clear visual representation of financial activity.

---

## Visual Goal Progress Dashboard

The dashboard displays:

* Total monthly spending
* Budget goal information
* Progress indicators
* Overspending notifications

This allows users to instantly understand their financial position.

---

# Gamification Feature

## Badge Achievement System

SMART BUDGET includes a gamification system designed to motivate positive budgeting behaviour.

Users can earn achievement badges for actions such as:

* Adding expenses
* Maintaining budget goals
* Demonstrating consistent financial tracking

The badge system encourages engagement and rewards responsible money management.

---

# Custom Feature 1

## Receipt Photo Management

Users can attach receipt images when recording expenses.

Benefits include:

* Proof of purchase storage
* Easier expense verification
* Improved financial record keeping

Users can later view uploaded receipt images from the expense history screen.

---

# Custom Feature 2

## Category Spending Limit Monitoring

Users can define spending limits for individual categories.

Examples:

* Food Limit = R1000
* Transport Limit = R500

The application automatically monitors spending against these limits.

When spending exceeds a category limit, SMART BUDGET displays an overspending warning to notify the user.

This feature promotes better financial discipline and assists users in identifying problematic spending behaviour.

---

# Database

SMART BUDGET uses SQLite for local data storage.

The database stores:

* User accounts
* Categories
* Expenses
* Monthly goals
* Spending limits
* Badge achievements
* Receipt image references

All data is stored locally on the device.
