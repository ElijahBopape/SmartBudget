# SMART BUDGET

## Overview

SMART BUDGET is an Android budgeting application developed for OPSC6311.

The application helps users manage personal finances through expense tracking, budgeting tools, spending reports, receipt storage, visual spending graphs, category limits, and gamification rewards.

## Features

### User Management

* User registration
* User login
* Secure local user profiles

### Expense Management

* Add expenses
* Expense categories
* Expense descriptions
* Expense dates
* Receipt image attachments

### Budget Management

* Monthly minimum goals
* Monthly maximum goals
* Category spending limits
* Overspending alerts

### Reports

* Expense viewing by period
* Category totals
* Spending summaries

### Dashboard

* Monthly spending overview
* Goal progress tracking
* Overspending notifications
* Badge summary

### Gamification

* Achievement badges
* Goal achievement rewards

### Custom Features

1. Payment Reminder System
2. Category Limit Monitoring with Overspending Alerts

## Database

SQLite local database is used to store:

* Users
* Categories
* Expenses
* Goals
* Badges
* Reminders
* Category limits

## Technologies Used

* Kotlin
* Android Studio
* SQLite
* MPAndroidChart
* Android SDK
